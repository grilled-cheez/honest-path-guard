# Honest Path Guard (VIGILANCE)

Zero-trust road inspection demo app with:
- **Field Worker** flow: NFC verification, photo capture simulation, interactive road-condition checklist, and on-chain inspection submission
- **Officer Dashboard**: GIS map view, policy violation feed, and immutable ledger audit trail
- **Light/Dark mode** toggle

## Live demo
`https://honest-path-guard-qyk8txled-grilled-cheezs-projects.vercel.app/`

## Run locally
### Prerequisites
- **Node.js 20 LTS** (recommended)

### Install & start
```bash
cd "/Users/pransan/Desktop/mtech/pravi_prob/honest-path-guard"
npm install
npm run dev
```

Then open `http://localhost:8080`.

### Build & preview (optional)
```bash
npm run build
npm run preview
```
